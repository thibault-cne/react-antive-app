## Installation guide

``` shell
git clone https://github.com/thibault-cne/react-native-app
cd react-native-app
```

``` shell
npm install
npm start
```

You can then follow ```expo``` indications to run on devices or on the web.
You can refer to the expo documentaion [here](https://docs.expo.dev).
